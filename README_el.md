# Mελέτες δεδομένων

Μελέτες αναπαραγώγιμων δεδομένων: ανάλυση μετανάστευσης/πληθυσμού και σύνολα δεδομένων εταιρειών (σημειωματάρια, αποτελέσματα, μέθοδοι). Κωδικός: MIT · Δεδομένα/έξοδοι: CC BY 4.0 (βλ. DATA_LICENSE.md)

## Τι υπάρχει μέσα
- **Μετανάστευση / Πληθυσμός**: μελέτες + σημειωματάρια + αναφορές.
- **Σύνολα δεδομένων εταιρείας**: καθαρισμός, EDA, μοντελοποίηση, KPI.
- **Μέθοδοι**: επαναχρησιμοποιήσιμα σενάρια/πρότυπα για αναπαραγώγιμη ανάλυση.

## Γρήγορη Έναρξη
1) Δημιουργήστε ένα περιβάλλον (Python/R)
2) Εγκαταστήστε deps
3) Εκτελέστε το/τα σημειωματάριο/α μελέτης στο `notebooks/` ή τα scripts στο `src/`

## Χάρτης Αποθετηρίου
- `data/` – ακατέργαστα/ενδιάμεσα/επεξεργασμένα σύνολα δεδομένων (δείτε την πολιτική δεδομένων παρακάτω)
- `notebooks/` – EDA, μοντελοποίηση, σημειωματάρια αναφοράς
- `src/` (ή `scripts/`) – επαναχρησιμοποιήσιμος κώδικας (ingest/clean/features)
- `reports/` – εξαγόμενες αναφορές HTML/PDF
- `figures/` – figures που χρησιμοποιούνται σε αναφορές/README
- `references/` – πηγές, σύνδεσμοι, σημειώσεις
