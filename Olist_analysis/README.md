# Olist — Exploratory Analysis (Customers & Closed Deals)

## Περιεχόμενο
Το repo περιέχει δύο ανεξάρτητες αναλύσεις (δεν υπάρχει κοινό κλειδί με τα διαθέσιμα αρχεία για join):

- **Customer Scope (B2C / Geography)**  
  Γεωγραφική κατανομή customer records ανά πολιτεία/πόλη (proxy ζήτησης).  
  → [Customer_Scope.md](./Customer_Scope.md)

- **Closed Deals (B2B / Sales Funnel)**  
  Ανάλυση κλεισμένων deals (won_date), κατανομή ανά μήνα και segmentation (business_segment, lead_type κ.λπ.).  
  → [closed_deals.md](./closed_deals.md)

## Outputs / Artifacts
Κάθε report έχει συνοδευτικά artifacts (CSV tables & PNG charts) στους αντίστοιχους φακέλους `outputs/`
και ενσωματώνεται στο markdown με relative links.
